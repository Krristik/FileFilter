использование:
  java -jar FileFilter.jar [опции] файл1 [файл2 ...]

опции:

  -o <путь> - выходная директория (по умолчанию: текущая)

  -p <префикс> - префикс для выходных файлов (по умолчанию: пустой)

  -s - вывести краткую статистику

  -f - вывести полную статистику

примеры:

  java -jar FileFilter.jar -s data1.txt data2.txt

  java -jar FileFilter.jar -o out -p sample_ -f *.txt

  java -jar FileFilter.jar -a log.txt

  java -jar FileFilter.jar input.txt

особенности:

1) Поддержка чисел в экспоненциальной форме (1.528535047E-25)
2) Точная арифметика через BigDecimal (без потери точности)
3) Обработка ошибок без падения программы
4) Кодировка UTF-8 для всех операций ввода-вывода
